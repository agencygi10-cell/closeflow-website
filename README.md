# CloseFlow System — Website

Landing page oficial de **CloseFlow System**, el sistema CRM para estudios de tatuaje y centros de belleza que quieren escalar.

> _"CloseFlow: El partner que organiza tu caos y multiplica tus ventas"_

## Stack

- [Next.js 14](https://nextjs.org/) (App Router) + TypeScript
- [Tailwind CSS](https://tailwindcss.com/) con paleta brand custom (morado / lila / rosa / negro)
- [Framer Motion](https://www.framer.com/motion/) para animaciones
- [React Hook Form](https://react-hook-form.com/) para el formulario de captura
- [Lucide](https://lucide.dev/) para iconos

## Secciones

1. **Hero** — Logo, tagline y CTA principal
2. **¿Qué hacemos?** — Sistema CRM explicado en 6 pilares
3. **Servicios** — GoHighLevel, Meta Ads, asesorías de contenido y para escalar
4. **Casos de éxito** — Stats, testimonios y link al drive de videos
5. **Cómo funciona** — Pasos + dashboard preview
6. **Formulario de captura** — Lead form (nombre, email, teléfono, mensaje)
7. **Footer** — Navegación, contacto y redes

## Desarrollo local

```bash
npm install
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000).

## Build de producción

```bash
npm run build
npm run start
```

## Deploy en Vercel

1. Entra a [vercel.com/new](https://vercel.com/new) e importa el repositorio.
2. Vercel detecta Next.js automáticamente — deja la configuración por defecto.
3. Click en **Deploy**.
